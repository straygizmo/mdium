# キーボードショートカット

## グローバル

| ショートカット | 操作                       |
| -------------- | -------------------------- |
| Ctrl+S         | 保存                       |
| Ctrl+Shift+S   | 名前を付けて保存           |
| Ctrl+Z         | 元に戻す                   |
| Ctrl+Y         | やり直し                   |
| Ctrl+F         | 検索                       |
| Ctrl+H         | 置換                       |
| Ctrl+T         | ターミナルの切替           |
| Ctrl+D         | フォルダを開く             |
| Ctrl+O         | opencode 設定を開く        |
| Ctrl+W         | タブを閉じる               |
| Ctrl+\\        | エディタパネルの切替       |
| Ctrl+\`        | ターミナルの切替           |
| Ctrl+Shift+R   | RAG パネルの切替           |
| Ctrl+Shift+C   | リッチテキストとしてコピー |
| Ctrl+B         | 太字（プレビューモード）   |
| Ctrl+I         | 斜体（プレビューモード）   |

## 検索・置換

| ショートカット | 操作                       |
| -------------- | -------------------------- |
| F3             | 次の検索結果               |
| Shift+F3       | 前の検索結果               |
| Enter          | 次の検索結果（検索入力欄） |
| Ctrl+Enter     | すべて置換                 |
| Escape         | 検索パネルを閉じる         |

## ファイルエクスプローラー

| ショートカット | 操作       |
| -------------- | ---------- |
| F2             | 名前変更   |
| Delete         | 削除       |
| Ctrl+X         | 切り取り   |
| Ctrl+C         | コピー     |
| Ctrl+V         | 貼り付け   |

## テーブルエディタ

| ショートカット           | 操作                         |
| ------------------------ | ---------------------------- |
| 矢印キー                | セル移動                     |
| Tab / Shift+Tab          | 次 / 前の列に移動            |
| Enter / F2               | セル編集開始                 |
| Escape                   | セル編集キャンセル           |
| Delete                   | セル内容のクリア             |
| Ctrl+Z / Ctrl+Y         | 元に戻す / やり直し          |
| Ctrl+C / Ctrl+X / Ctrl+V| コピー / 切り取り / 貼り付け |
| Ctrl+B                   | 太字                         |
| Ctrl+I                   | 斜体                         |
| Ctrl+5                   | 取り消し線                   |
| Ctrl+\`                  | コード                       |

## 画像キャンバスエディタ

| ショートカット         | 操作           |
| ---------------------- | -------------- |
| Ctrl+Z                 | 元に戻す       |
| Ctrl+Y / Ctrl+Shift+Z | やり直し       |
| Delete / Backspace     | 選択要素の削除 |

## エディタ

| ショートカット | 操作           |
| -------------- | -------------- |
| Tab            | 2スペース挿入  |

## AI 生成モーダル

| ショートカット | 操作             |
| -------------- | ---------------- |
| Ctrl+Enter     | 生成実行         |
| Escape         | モーダルを閉じる |
