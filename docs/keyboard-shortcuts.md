# Keyboard Shortcuts

## Global

| Shortcut         | Action                    |
| ---------------- | ------------------------- |
| Ctrl+S           | Save                      |
| Ctrl+Shift+S     | Save As                   |
| Ctrl+Z           | Undo                      |
| Ctrl+Y           | Redo                      |
| Ctrl+F           | Find                      |
| Ctrl+H           | Replace                   |
| Ctrl+T           | Toggle terminal           |
| Ctrl+D           | Open folder               |
| Ctrl+O           | Open opencode config      |
| Ctrl+W           | Close tab                 |
| Ctrl+\\          | Toggle editor panel       |
| Ctrl+\`          | Toggle terminal           |
| Ctrl+Shift+R     | Toggle RAG panel          |
| Ctrl+Shift+C     | Copy as rich text         |
| Ctrl+B           | Bold (preview mode)       |
| Ctrl+I           | Italic (preview mode)     |

## Search & Replace

| Shortcut     | Action                        |
| ------------ | ----------------------------- |
| F3           | Next search result            |
| Shift+F3     | Previous search result        |
| Enter        | Next result (in search input) |
| Ctrl+Enter   | Replace all                   |
| Escape       | Close search panel            |

## File Explorer

| Shortcut | Action |
| -------- | ------ |
| F2       | Rename |
| Delete   | Delete |
| Ctrl+X   | Cut    |
| Ctrl+C   | Copy   |
| Ctrl+V   | Paste  |

## Table Editor

| Shortcut                 | Action                |
| ------------------------ | --------------------- |
| Arrow keys               | Navigate cells        |
| Tab / Shift+Tab          | Next / previous column|
| Enter / F2               | Start editing cell    |
| Escape                   | Cancel cell edit      |
| Delete                   | Clear cell content    |
| Ctrl+Z / Ctrl+Y         | Undo / Redo           |
| Ctrl+C / Ctrl+X / Ctrl+V| Copy / Cut / Paste    |
| Ctrl+B                   | Bold                  |
| Ctrl+I                   | Italic                |
| Ctrl+5                   | Strikethrough         |
| Ctrl+\`                  | Code                  |

## Image Canvas Editor

| Shortcut               | Action                |
| ---------------------- | --------------------- |
| Ctrl+Z                 | Undo                  |
| Ctrl+Y / Ctrl+Shift+Z | Redo                  |
| Delete / Backspace     | Delete selected items |
