## Highlights

MDium v0.2.0 is the first public release, establishing MDium as a Markdown-centric document foundation for the AI era. This release includes a full-featured Markdown editor with rich import/export capabilities, AI-powered document workflows, and seamless integration with external platforms.

---

## Major Changes

### 1. Slidev Presentation System
Markdown files can now be previewed and exported as slide presentations using Slidev. The integrated preview panel displays slides in real-time via an iframe, with support for fullscreen mode, theme selection, and export to PDF/PPTX formats. A bundled Slidev environment ensures zero-configuration setup.

### 2. Video Generation from Markdown
A complete Markdown-to-video pipeline has been implemented. Features include:
- LLM-powered narration and scenario generation
- VOICEVOX TTS with per-segment audio, volume, and speed controls
- SRT subtitle generation with captions overlay
- Background effects (gradient, particles, Three.js geometry, Lottie animations)
- Scene-by-scene editing UI with undo/redo support
- MP4 export via Playwright rendering + FFmpeg encoding

### 3. MCP (Model Context Protocol) Integration
Built-in MCP server support enables AI-powered image generation directly from the editor. Features include a configurable MCP server manager in settings, a `mcp_call_tool` command for invoking tools via stdio, and a context menu for generating images within documents.

### 4. XLSX/XLSM to Markdown Conversion
Excel files (.xlsx/.xlsm) can be converted to Markdown tables with a single click in the preview panel. The vendored xlsx2md library handles complex spreadsheets including merged cells, with image assets extracted to a dedicated folder.

### 5. VBA Macro Export/Import
Full VBA macro round-trip support for Excel files (.xlsm/.xlam). Implements MS-OVBA 2.4.1 spec-compliant compression/decompression, enabling extraction of VBA modules to editable text files and re-injection back into Excel files.

### 6. Monaco Code Editor Integration
An integrated Monaco Editor provides syntax-highlighted editing for code files (`.js`, `.ts`, `.py`, `.rs`, etc.) with language auto-detection, enabling MDium to function as a lightweight code editor alongside its document capabilities.

### 7. RAG Agent System
Each workspace runs its own opencode-sdk instance with a built-in RAG search tool. The agent system supports file-based agent definitions with a markdown editor, global/project scope management, and a selectable agent dropdown in the chat interface.

### 8. Speech Recognition & Voice Input
Voice input via `Ctrl+Shift+M` with support for the Moonshine speech recognition model. Improved reliability for speech-to-text workflows, enabling hands-free document authoring.

### 9. Zenn Article Workflow
Integrated support for the Zenn publishing platform, including auto-detection of Zenn projects, frontmatter editing in the preview panel, folder initialization, and built-in commands for article creation, proofreading, and deployment.

### 10. Medium Publish Feature
Publish Markdown articles directly to Medium from within MDium. Includes API token configuration, a publish dialog with title/tags/status selection, and integration with the preview panel toolbar.

### 11. External File Change Detection
When files are modified outside MDium, a diff dialog shows the changes with a side-by-side view, allowing users to accept external changes, keep the current version, or review the differences before deciding.

### 12. Restore Last Folders on Startup
Optionally restore previously opened workspace folders when launching MDium. Folder paths are persisted via Zustand middleware, with a toggle in Settings to enable/disable this behavior.

### 13. Scope UI Redesign
Settings sections (Agents, Custom Tools, Skills, MCP Servers, Commands, Rules) now use a unified list with a Global/Project scope toggle, replacing the previous tab-based approach. This provides a cleaner, more consistent configuration experience.

### 14. Built-in Command System
A command dropdown in the preview toolbar enables one-click execution of built-in commands such as Slidev presentation generation, KityMinder mind map conversion, and video scenario creation, with support for template parameters and custom command registration.

### 15. i18n & Quality Improvements
- Full internationalization: all UI-facing strings use i18n translation keys (English/Japanese)
- All code comments translated to English
- Unsaved changes confirmation dialog on app close
- Auto-scroll tab bar to active tab
- CSV export from table context menu in preview
- Configurable MCP server timeout

## Technical Stack

- **Frontend:** React + TypeScript + Zustand + Vite
- **Backend:** Tauri (Rust)
- **AI Integration:** opencode-sdk, MCP protocol
- **Video:** open-motion (Remotion-based), VOICEVOX, FFmpeg
- **Presentation:** Slidev v51
- **Code Editor:** Monaco Editor
