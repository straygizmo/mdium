# Project Instructions

## Superpowers

Save all superpowers plans and specs to `.superpowers/` instead of `docs/superpowers/`.

- Plans: `.superpowers/plans/YYYY-MM-DD-<feature-name>.md`
- Specs: `.superpowers/specs/YYYY-MM-DD-<feature-name>-design.md`

## Coding Standards

- All code comments must be written in English.
- Never hardcode UI-facing strings (button labels, messages, etc.). Always use i18n for all user-visible text.
